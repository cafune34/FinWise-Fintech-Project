# FinWise V2 - Ekran Görüntüleri (Screenshots)

Bu klasör, FinWise V2 projesinin final arayüz ekran görüntülerinin jüriye sunumu ve rapor içi görselleştirilmesi amacıyla ayrılmıştır. 

Lütfen uygulamayı yerel ortamda (`npm run dev`) çalıştırıp aşağıdaki sayfaların ekran görüntülerini alarak bu klasöre kaydedin:

1. `landing.png` - `/` Landing / Tanıtım Sayfası
2. `dashboard.png` - `/dashboard` Genel Bakış Paneli
3. `accounts.png` - `/accounts` Banka Hesapları Sayfası
4. `transactions.png` - `/transactions` İşlem Geçmişi Sayfası
5. `budget.png` - `/budget` Bütçe Limitleri ve Harcama Tahminleme Sayfası
6. `risk-monitoring.png` - `/regtech` Risk İzleme / RegTech Sayfası
7. `payments.png` - `/payments` Ödeme Talimatları Sayfası
8. `investment-profile.png` - `/robo-advisor` Yatırım Profili / Robo-Danışmanlık Sayfası

*Not: Görüntülerin masaüstü ölçekte (tercihen 1366x768 veya 1440x900 çözünürlüklerinde) ve koyu tema (dark mode) renklerinin net okunabileceği şekilde çekilmesi önerilir.*
