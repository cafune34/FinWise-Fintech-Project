# Sprint 6 - UI Redesign ve Türkçe Metin Temizliği

## Sprint amacı

FinWise arayüzünü masaüstü webde profesyonel görünen kurumsal fintech paneli seviyesine taşımak, Türkçe metinleri düzeltmek ve kullanıcıya görünen eski sprint/deneme dilini kaldırmak.

## Yapılan işler

- AppShell modern sidebar, üst bar, aktif menü ve kullanıcı özetiyle yenilendi.
- Landing page FinWise ürün vitrini olarak yeniden tasarlandı.
- Dashboard finansal sağlık skoru, özet metrikler, hızlı işlem kartları, hesap özeti, grafikler ve akıllı uyarılarla güçlendirildi.
- Hesaplar, İşlemler, Bütçe Planı, Ödeme Talimatları, Risk İzleme ve Yatırım Profili sayfaları yeniden düzenlendi.
- Recharts kullanan grafiklerde ölçü alanları daha deterministik hale getirildi.

## UI değişiklikleri

- Koyu lacivert/antrasit zemin ve cyan/turkuaz vurgu dili standartlaştırıldı.
- Kart, tablo, filtre ve form alanları masaüstü kullanımına uygun hale getirildi.
- Sidebar daha dolu, kurumsal ve ikon destekli yapıya taşındı.
- Dashboard içerik genişliği ve spacing 1366 px / 1440 px sunum ekranları için dengelendi.

## Türkçe metin temizliği

- Dashboard yerine Genel Bakış kullanıldı.
- Bütçe, İşlemler, Ödeme, Şirket, Nakit Akışı, Yatırım Profili ve Risk İzleme yazımları Türkçe karakterli hale getirildi.
- RegTech görünen isim olarak Risk İzleme diline çekildi.
- Robo Danışman görünen isim olarak Yatırım Profili diline çekildi.
- Risk seviyeleri Yüksek, Orta ve Düşük olarak gösterildi.

## Kabul kriterleri

- [x] UI metinleri Türkçe karakterli ve ürün odaklı.
- [x] Eski sprint/deneme dili kullanıcı arayüzünden kaldırıldı.
- [x] Landing page ve Dashboard profesyonel fintech görünümüne taşındı.
- [x] Modül sayfalarında masaüstü okunabilirliği artırıldı.
- [x] `npm run lint` başarılı olmalı.
- [x] `npm run build` başarılı olmalı.
