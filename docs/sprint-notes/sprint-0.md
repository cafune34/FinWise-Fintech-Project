# Sprint 0 Notlari

## Sprint amaci
FinWise projesi icin temiz bir proje iskeleti kurmak, temel dokumantasyonu hazirlamak ve gelistirme surecini sprint duzeninde baslatmak.

## Yapilan isler
- Repo branch ve remote yapisi dogrulandi.
- README, FinWise ve ders baglamina gore guncellendi.
- AGENTS.md, kod ajanlari icin proje kurallariyla guncellendi.
- `src/components`, `src/data`, `src/lib`, `src/types` klasorleri olusturuldu.
- Ana sayfa minimal FinWise baslangic metniyle sade hale getirildi.

## Kullanilan teknolojiler
- Next.js App Router
- TypeScript
- Tailwind CSS

## Kabul kriterleri
- Dokumantasyon dosyalari eksiksiz ve guncel.
- Sprint 0 kapsaminda yalnizca proje iskeleti hazirlandi.
- Build komutu basariyla tamamlandi.

## Sonraki sprintte yapilacaklar
- Temel ekran akislari ve bilgi mimarisi taslagi
- Mock veri modellerinin detaylandirilmasi
- Ilk fonksiyonel UI bilesenlerinin eklenmesi

## Not
Proje egitim amaclidir, gercek finansal islem yapilmaz.
